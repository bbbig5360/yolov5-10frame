- YOLOv5 기법을 이용해 사람만 검출하는 인공지능 모델을 만들었습니다.

  이유 : YOLO특성상 작은 사람과 겹친 사람을 검출하지 못해서입니다.
  
- 해당 소스를 제공받았을 때 하나의 동영상(또는 이미지)을 디텍팅했습니다.
